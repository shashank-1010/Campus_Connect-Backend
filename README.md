# Campus Connect

A full-stack MERN application for campus students.

## Setup

1. Fill in `.env` with your MongoDB URI:
```
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=campusconnectsecret
```

2. Install all dependencies:
```
npm install
```

3. Start development (runs server + client together):
```
npm run dev
```

- Server runs on http://localhost:5000
- Client runs on http://localhost:3000
